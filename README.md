# NutriMind AI 🧠🥗

An AI-powered nutrition, meal cost & food optimisation platform.

## Stack
- **Backend:** FastAPI, PostgreSQL, SQLAlchemy
- **Data Science:** pandas, scikit-learn, XGBoost, Prophet, OR-Tools
- **Frontend:** React, Tailwind CSS, Recharts (via Lovable)

## Features
- Food search via OpenFoodFacts API
- Nutrition efficiency scoring (Protein Efficiency Score)
- Meal cost analytics & dashboards
- AI-powered meal plan optimisation
- Behavioural analytics & anomaly detection
- Predictive nutrition forecasting
